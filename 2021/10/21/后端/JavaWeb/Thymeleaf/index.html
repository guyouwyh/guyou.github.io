# thymeleaf 简介

现代化、服务端 Java 模板引擎

# 基本语法

# 表达式

表达式名字语法用途
变量取值${...}获取请求域、session 域、对象等值
选择变量*{...}获取上下文对象值
消息#{...}获取国际化等值
链接@{...}生成链接
片段表达式~{...}jsp:include 作用，引入公共

引入依赖

在 html 页面中添加头文件

# 变量取值

用于取出从业务层传来的 Model 或 ModelAndView 对象中的键值。

# 选择变量

在使用选择变量之前，必须有 th:object 先绑定一个对象，然后通过 *{对象属性名} 进行取值。

# 链接

@{……} 支持决定路径和相对路径。其中相对路径又支持跨上下文调用 url 和协议的引用

# 常用的 th 标签

#dates 日期转换

将后台传入到前端的 Date 数据类型转换为指定格式的日期。

text untext 转义和非文本转义

th:text 不会将字符转义

th:untext 会将字符转换为 html

迭代

msg 是从后台传过来的集合类型，it 是集合中的一个对象，statu 中的属性可以显示迭代的次数等

条件判断

如果 msg 不等于 null 才会显示这条语句

定义模板片段

可在其他页面中使用该片段

语法：~{片段所在的页面的名称：：定义片段时的名称}

th:insert ：将被引用的模板片段插⼊到自己的标签体中
th:replace ：将被引用的模板片段替换掉自己
th:include ：类似于 th:insert，⽽不是插⼊⽚段，它只插⼊此⽚段的内容。

th:attr 设置属性值
