# 入口函数

原生 js 的入口函数和 Jquery 的入口函数的区别

  • 加载模式的不同：

    • 原生 js 会等到 Dom 元素加载完毕，并且图片也加载完毕才会执行
    • Jquery 会等到 Dom 元素加载完毕，但不会等到图片加载完毕

  • 编写多个入口函数，jQuery 后面的不会覆盖前面的

暂停入口函数的执行

holdReady() ：true 暂停，false 不暂停

# 基本选择器

JQuery 选择器类似于 css,css 有 id 选择器用 #id , 类选择器 .class .

id 选择器

JQuery 的 id 选择器是利用 DOM 元素的 id 属性值来筛选匹配的元素，并以 JQuery 包装集的形式返回给对象。

语法: ${#id名称} 获取对应 id 的元素

元素选择器

元素选择器是根据元素名称匹配相应的元素。元素选择器指向的是 DOM 元素的标记名，也就是说元素选择器是根据元素的标记名选择的。

语法: ${元素标签名称}

类名选择器

选择器是通过元素拥有的 CSS 类的名称查找匹配的 DOM 元素。在一个页面中，一个元素可以有多个 CSS 类，一个 CSS 类又可以匹配多个元素，如果有元素中有一个匹配类的名称就可以被类选择器选取到。简单地说类名选择器就是以元素具有的 CSS 类名称查找匹配的元素。

语法： ${.class名称}

# 属性选择器

​ 属性选择器就是通过元素的属性作为过滤条件进行筛选对象

[attribute]

​ 说明：匹配包含给定属性的元素

​ 示例：$("div [name]") // 匹配包含有 name 属性的 div 元素

​ [attribute=value]

​ 说明：匹配属性值为 value 的元素

​ 示例：$("div [name='test']") // 匹配 name 属性是 test 的 div 元素

​ [attribute!=value]

​ 说明：匹配属性值不等于 value 的元素

​ 示例：$("div [name!='test']") // 匹配 name 属性不是 test 的 div 元素

​ [attribute*=value]

​ 说明：匹配属性值含有 value 的元素

​ 示例：$("div [name*="test"]") // 匹配 name 属性值中含有 test 值的 div 元素

​ [attribute^=value]

​ 说明：匹配属性值以 value 开始的元素

​ 示例：$("div [name^='test']") // 匹配 name 属性以 test 开头的 div 元素

​ [attribute$=value]

​ 说明：匹配属性值以 value 结束的元素

​ 示例：("div[name("div[name("div[name='test']") // 匹配 name 属性以 test 结尾的 div 元素

# Dom 操作

# 获取内容

三个简单实用的用于 DOM 操作的 jQuery 方法：

  • text () - 设置或返回所选元素的文本内容
  • html () - 设置或返回所选元素的内容（包括 HTML 标记）
  • val () - 设置或返回表单字段的值

# 获取 / 添加属性

attr (""): 用于获取 / 设置具体属性值。

注意点：无论获取多少的节点，都只获取第一个节点的属性值，

如果是设置属性，设置所有节点的属性值

prop 同理

prop 和 attr 的区别:

  • prop 如果没有相应的属性，返回空字符串，attr 如果没有相应的属性，返回 undefined

  • 对于 HTML 元素本身就滴啊有的属性，一般使用 prop 方法

  • 对于自定义的属性，一般使用 attr 方法。

  • 一般具有 true、false 值的属性节点，例如 selected、checked 或者 disabled 都使用 prop

# 删除属性

removeAttr ('class'): 删除属性节点

注意：如果要删除的多个属性节点，以空格隔开

prop 同理

$("<li>13213</li>") ：创建一个元素

# 添加元素 / 内容

  • append () - 在被选元素的结尾插入内容
  • prepend () - 在被选元素的开头插入内容
  • after () - 将元素添加到指定元素外部的最后
  • before () - 将元素添加到指定元素外部的首行
  • appendTo (): 将元素添加到某个节点内容的最后一行
  • prependTo (): 将元素添加到某个节点内容的首行

# 删除元素 / 内容

如需删除元素和内容，一般可使用以下两个 jQuery 方法：

  • remove () - 删除被选元素（及其子元素）
  • empty () - 从被选元素中删除子元素

# 替换元素

  • replaceWith(content) ; 替换所有匹配的节点

# 复制节点

  • clone (参数): 参数为 false 是浅复制，参数为 true 是深复制

浅复制：只会复制元素，不会复制元素的事件

深复制：不仅会复制元素，还会复制元素的事件

# 操作 CSS

jQuery 拥有若干进行 CSS 操作的方法。我们将学习下面这些：

  • addClass () - 向被选元素添加一个或多个类
  • removeClass () - 从被选元素删除一个或多个类
  • toggleClass () - 对被选元素进行添加 / 删除类的切换操作
  • css () - 设置或返回样式属性，如果后面是数值，可以不加单位，参数可以以对象形式

# 遍历元素

  • $("").each(function(index,domEle)) :index 是每个元素的索引号，domEle 是每个 DOM 元素对象
  • $.each(arr,function(index,value)) : 遍历数组和对象，第一个参数为要遍历的数组，第二个函数中 index 是索引号，value 是指

# Jquery 的遍历

$.each(obj,function(index,value)) : 可以遍历伪数组或者是数组

$.map(obj,function(index,value))

# 向上遍历 DOM 树

  • parent ()：parent () 方法返回被选元素的直接父元素。

    该方法只会向上一级对 DOM 树进行遍历。

  • parents ()：parents () 方法返回被选元素的所有祖先元素，它一路向上直到文档的根元素 html 。

  • parentsUntil ():parentsUntil () 方法返回介于两个给定元素之间的所有祖先元素。

# 向下遍历 DOM 树

向下遍历 DOM 树

  • children ():children () 方法返回被选元素的所有直接子元素。

    该方法只会向下一级对 DOM 树进行遍历。

  • find ():find () 方法返回被选元素的后代元素，一路向下直到最后一个后代。

  • siblings (): 返回指定类型的兄弟节点

# 尺寸和位置的操作

  • height() : 获取和设置元素的高度

  • width() : 获取和设置元素的宽度

  • offset() : 获取元素距离窗口的偏移位

    • left: 距离左边
    • right: 距离右边
    • top: 距离上边
    • bottom: 距离下边

    设置可以传入一个对象，里面包含上下左右的值。

  • position() ：获取元素距离定位元素的偏移位

    • left: 距离左边
    • right: 距离右边
    • top: 距离上边
    • bottom: 距离下边

    注意:position () 只能获取，不能设置

  • scrollTop : 获取元素滚动的偏移位

# 事件

事件绑定

on(事件触发方式,子元素触发,触发函数)

事件自动触发

triggle(触发方式) : 会触发事件冒泡

triggleHandler(触发方式) ：不会触发元素的默认行为，不会触发事件冒泡

事件的解绑

off("解绑的事件类型","事件处理程序") : 如果没有填参数就解绑所有事件

自定义事件

必须满足两个条件

  • 事件必须通过 on 绑定
  • 事件必须通过 triggle 触发

# 动画

  • show(动画执行的时间,执行完之后的回调函数) : 显示
  • hide(动画执行的时间,执行完之后的回调函数) : 隐藏
  • toggle(动画执行的时间,执行完之后的回调函数) : 切换
  • slideDown : 展开
  • slideup : 收起
  • stop(是否停止后续动画,是否继续当前动画) : 停止当前动画，默认参数为 false

注意：在 jQuery 执行动画之前先调用 stop 方法，然后再执行动画

淡入淡出效果

fadeIn : 淡入

fadeOut : 淡出

fadeToggle : 切换

fadeTo(执行时间,淡入的程度[.2],回调函数) :

自定义动画

animate(参数对象,持续时间,动画执行曲线[linear],回调函数)

delay(延迟动画的时间)
