# 网页的组成

  • HTML：用于描述页面的结构
  • CSS：用于控制页面的元素的样式
  • JavaScript：用于响应用户的操作

# HTML 简介

HTML（ Hypertext Markup Language ） 超文本标记语言，它使用标签的形式来标识网页中的不同组成部分。其中，超文本可以指超链接，使用超链接可以让我们从一个页面跳转到另一个页面。

# 标签中的属性

属性描述

id定义元素的唯一 id
style规定元素的行内样式（inline style）

# 转义字符

在文档中，有些特殊符号并不能直接书写（多个连续空格、小于 / 大于号），此时需要实体（转义字符），其基本语法为 &实体名; 。如下：

  • ：表示不间断空格 (non-breaking space), &nbsp;
  • > ：表示大于号， &lt;
  • < ：表示小于号， &gt;
  • × ：表示乘号
  • " ：表示引号

# 文本格式化

  • <h1-h6> 标题标签：h1~h6 共六级标题，属于块元素。

    <hgroup> 标签能够为标题标签分组，即将一组相关的标题同时放入到 hgroup 标签中。（html5）

  • <p> ：表示页面中的一个段落，它会独占一行，属于块元素。

  • <em> ：语音语调加重，它的样式也许是斜线样式，属于行内元素；

    <strong> ：强调重要内容，属于行内元素。

  • <blockquote> ：表示一个长引用，属于块元素；

    <q> ：表示短引用，不会独占一行！

  • <br> ：换行

  • <hr> ：展示一条水平线

# 上标标签和下标标签

上标标签

sup : 标签中的内容会偏向右上角

下标白浅

sub : 标签中的内容会偏向右下角

# 区块布局

div : 布局元素，属于块内元素，无具体特定语义，用于组合其他 HTML 的容器，使用频率极为频繁。

span ：用于在网页中选中文字，属于行内元素，无具体特定语义，可用作文本的容器。

# 列表

  • 有序列表：使用 <ol> 来创建，使用 <li> 表示列表项，几个 <li> 表示共有几个项。 type 属性可以指定排列方式，例如从 a 或 I 开始。

  • 无序列表：使用 <ul> 来创建，使用 <li> 表示列表项，几个 <li> 表示共有几个项。多用于制作网页上的菜单。 type 属性可以用来定义前面的样式，空心圆 circle

  • 自定义列表： <dl> 用于定义描述列表，与 <dt> (定义项目 / 名字) 和 <dd> (描述每一个项目 / 名字) 一起使用。

# 表格

  • 表格由 <table> 标签来定义，其常用属性有：

    • border : 表示边框的宽度
    • bordercolor ：表格边框的颜色
    • cellspacing ：单元格之间的间隔
    • cellpadding : 单元格的内边距
    • width ：表格的总宽度
    • height ：表格的总高度
    • align ：表格整体的对齐方式（left,center,right）

  • bgcolor: 表格的整体颜色

  • 每个表格均有若干行，由 <br> 标签定义

    • bgcolor : 行的颜色
    • align : 行内文字的水平对齐方式

  • 每行被分隔成若干个单元格，由 <td> 标签定义， td 即 table data，其常用属性有

    • width : 设置单元格的宽度，设置后对当前一列的单元格都有影响
    • height : 单元格的宽度，设置后对的当前一行的单元格都有影响
    • bgcolor : 设置单元格的背景颜色
    • align : 单元格文字的水平对齐方式
    • rowspan : 合并垂直方向的水平单元格
    • colspan : 合并水平方向的单元格
    • valign : 单元格文字的垂直对齐方式

    表格结构标签

    • <caption> ：表格的标题

    • <thead> : 表示表格的头部区域

    • <tbody> : 表示表格的主题区域

    • <tfoot> : 表示表格的页尾区域

    合并单元格三部曲

    1. 先确定跨行还是跨列

    2. 找到目标单元格，写上合并方式 = 合并的单元格数量，如果是跨行合并就用 rowspan , 如果是跨列合并就用 colspan . 遵循左上原则，跨行在最上方的单元格加标签，跨列在最左边的单元格加标签

    3. 删除多余单元格

# 框架集

frameset : 设置框架 `

  • rows ：设置行数
  • cols : 设置列数

frame : 嵌套在 frameset 中

  • src : 引用外部页面
  • name : 可以在外部页面中使用 a 标签中的 target 值与之保持一致实现跳转

缺陷：不能写在 body 中，对爬虫不友好，对搜索引擎不友好。

iframe : 内联框架，可以写在 body 中.

# 超链接

标签 <a> ，属性有

  • href : 指定跳转的 url
    • 当需要跳转一个服务器内部路径时候
      • ./ 表示当前文件所在目录
      • ../ 表示当前文件所在目录的上一级目录
  • target : 用于指定
    • _self : 默认值，在当前页面中打开超链接
    • _blank : 在一个新的空白页面中打开超链接

# 表单

一个完整的表单通常由表单域、表单控件和提示信息三部分组成

使用 form 标签，其属性有

  • action : 指定提交数据的 url
  • method : 指定提交方式（post 和 get）

表单项标签，使用 <input> 标签

  • type 属性
    • placeholder : 属性，能够指定输入框的提示信息，当输入框的内容发生变化时候，会自动情况原来的提示信息。
    • value 属性：能够在文本框中填入 value 属性值，能够实现回显信息。若 value 属性与 readonly="readonly" 一起使用，则为该文本框所填的内容只读而不能修改。
    • password : 密码输入框
    • radio : 单选框，注意
      • 要想让多个单选框实现单选的效果，则多个单选框的 name 属性值必须一一
      • 一般会给每一个单选框提供 value 属性，指定其被选中后提交的值
    • checkbox : 复选框
    • file : 文件选择框
    • date : 包含年、月、日的控件
    • hidden : 隐藏域，用于提交一些信息
    • submit : 提交按钮
    • button : 普通按钮
    • image : 图片提交按钮， src 属性指定图片路径
  • name 属性：定义 input 属性的名称
  • checked 属性，input 元素首次被加载时应当被选中
  • maxlength : 输入字段中字符的最大长度

下拉列表标签 select , 其属性有

  • name 属性，提交后台时候的属性名

子元素 option 标签

  • value 属性，
  • selected 属性，默认选中这个带 selected 属性的标签

文本域标签 <textarea>

  • cols ：文本框长
  • rows : 宽所占的文字数

<laber> 标签：用于绑定一个表单元素，当点击 <label> 标签内的文本时，浏览器就会自动将焦点转到或者选择对应的表单元素上，用来增加用户体验

label 标签中的 for 属性值要和对应表单元素中的 id 属性值一样

filedset ：组合表单中的相关元素

legend :legend 元素为 filedset 元素定义标题（caption）
