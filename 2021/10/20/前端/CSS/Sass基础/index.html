# 数字函数

  • abs (): 绝对值
  • Round (): 四舍五入
  • ceil (): 向上取整
  • floor (): 向下取整
  • percentage (): 百分数
  • min (): 取最小数字
  • max (): 取最大的数字

# 字符串函数

  • to-upper-case (): 大写
  • to-lower-case (): 小写
  • str-length (): 获取字符串的长度
  • str-index (str,s): 判断字符串 s 的位置，从 1 开始

# 颜色函数

  • lighten (color,n%): 调亮
  • darken (color,n%): 调暗
